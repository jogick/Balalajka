Проект тестера, для прозвонки кабеля.
Тут выложен проект прошивки и схема принципиальная.

Принцип работы основан на смене полярности на проверяемом кабеле, со стороны приспособления. На противоположном конце обычный диод.

При подключении, через кабель, диода к приспособлению, диод светится, на приспособлении светится индикатор и включается звуковой сигнал.
